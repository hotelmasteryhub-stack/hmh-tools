WHY ONBOARDING NEEDS A SYSTEM

Most small hotels train new staff the same way — someone experienced works alongside the new person for a few shifts and they pick things up as they go. The problem is what they pick up depends entirely on who trained them, what that person remembered to show, and how busy they were that week.

Two receptionists trained this way may handle the same situation completely differently. Two housekeepers may clean to completely different standards. And as the manager, you have no way to know what was covered and what was missed — until something goes wrong in front of a guest.

A structured onboarding tracker does not make training more complicated. It makes it visible. You can see exactly where each person is in their training, what has been signed off, and what still needs to happen before they work alone.

THE ONBOARDING BUDDY — HOW IT WORKS

The buddy is not the manager checking in from a distance. The buddy is an experienced colleague who works alongside the new starter, shows them how things actually work at your hotel, and signs off each task when they are satisfied the trainee can do it independently.

This matters because the buddy's sign-off is a real confirmation — not a tick in a box. It means: I watched this person do this task. They can do it without help.

Brief your buddy before the new starter arrives

Show them the tool. Explain the four stages. Make clear that Signed Off means independent — not just observed. A good buddy makes onboarding faster, not slower.

The four training stages
1 — Observation
Trainee watches buddy do the task. No pressure to perform.
2 — Guided
Trainee does it with buddy present and supporting.
3 — Independent
Trainee does it alone. Buddy checks the result.
✓ Signed Off
Buddy confirms: this person can do this task without help.
This does not have to feel like extra work

The buddy was going to train the new person anyway. The tracker just makes it structured. After the first hire, the template is ready. The second hire takes half the setup time. The third is instant. And every person trained this way reaches independence faster — because nothing important gets skipped.

HOW TO USE THIS TOOL
1
Add a new trainee
Click + ADD TRAINEE in the tab bar. Choose their role — Housekeeper, Receptionist, or a custom role you name yourself. Each trainee gets their own tab with a full onboarding tracker.
2
Customise in Edit mode
Switch to Edit mode on the trainee's page. Add, remove, or rewrite tasks to match your hotel's standards. Edit the access checklist to reflect your systems. Do this once — the next trainee in the same role can duplicate the template.
3
Track in Use mode
Switch to Use mode to track the trainee's progress. Set the stage for each task as training happens. Add buddy sign-off and date. The progress bar updates automatically.
4
Track multiple people at once
Each trainee is a separate tab. Open three housekeeper tabs for three trainees starting in the same week. The Staff List tab shows everyone's progress at a glance.
5
Duplicate for the next hire
Once you have customised a template, duplicate it for the next person in that role. They start with your hotel's tasks already set up — you just fill in their name and start date.
6
Export when complete
Use the PDF, CSV, or Email buttons on each trainee's page for that person's record. Use the buttons in the top header to export all trainees at once.
EXPORT OPTIONS
PDF Print the full onboarding document

Per trainee — prints their profile, access checklist, all tasks with stages, and final sign-off. Top header PDF prints all trainees in one document.

CSV Export to spreadsheet

Per trainee — all tasks with current stage. All CSV exports every trainee together. Good for HR records or keeping a history of completed onboardings.

✉ EMAIL Send a progress update

Per trainee — progress summary with pending tasks. All Email sends a summary of all active trainees. Good for updating a property owner or manager who is off site.

BACKUP ↓ Save everything as a JSON file

Downloads a complete backup of all trainees — their profiles, access checklists, all tasks, and all progress stages. Found in the top header. Store this file somewhere safe. If your browser data is ever cleared or you switch devices, you can restore from this file and lose nothing.

RESTORE ↑ Load from a backup file

Select a .json backup file to restore all trainee data. You will be asked to confirm before anything is replaced — this action replaces all current data. Use it when switching devices or after a browser reset.
