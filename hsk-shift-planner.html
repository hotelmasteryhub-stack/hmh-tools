HSK Shift Planner
Room-by-room shift planning with real room numbers, configurable cleaning types, capacity tracking per staff member, and printable assignment sheets.
HOW IT WORKS — 4 STEPS
1
SETUP — do once, update when things change
Define your cleaning types (Departure, Stayover, Amenities Restock, Linen Change, etc.) with a default time. Set room types. Build the time matrix — each cleaning type × room type combination gets its own target time. Add your room list with real room numbers. Set up staff with default shift times. Add ad-hoc tasks.
2
ROOM BOARD — set each room's status every morning
Every room is a card. Select the cleaning type for today. The target time loads automatically from the matrix for that room type. Assign to a staff member. Add room-level tasks (e.g. crib setup, extra towels) — they stack on top of the cleaning time. Click AUTO-ASSIGN to distribute unassigned rooms evenly by workload.
3
CAPACITY — per staff and whole team
Each staff member has their own shift start and end time. Effective minutes = (end − start) × 60 × (1 − buffer%). A capacity bar shows assigned minutes vs effective minutes. Green = within capacity, amber = within 30 min over, red = overloaded. Team total shown across all staff.
4
PRINT — hand a sheet to each housekeeper
PRINT ALL prints every housekeeper's assignment sheet at once. PRINT THIS prints one person. Each sheet shows real room numbers, floor, cleaning type, target minutes (including room tasks), estimated finish, done ✓ column, and manager sign-off line.
THE TIME MATRIX — HOW CLEANING TIMES WORK

Cleaning time depends on two things: what type of clean and what type of room. A departure clean in a suite takes much longer than in a single room. The matrix in Setup lets you set specific times for every combination.

ONE RULE

A housekeeper without a written assignment decides for themselves what to clean and in what order. That is not a system — it is a daily gamble. Print the sheet. Hand it over. Every shift.
