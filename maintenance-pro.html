Maintenance Toolkit — Quick Start Guide
Hotel Mastery Hub · Maintenance Mastery Toolkit
WHAT THIS TOOLKIT DOES

This toolkit gives you everything you need to run maintenance in a small hotel — in one place. Log issues, run checklists, write procedures, manage suppliers, and assess how your system is working. You don't need to use all of it on day one. Start with what you need most.

Everything saves automatically in your browser. Your data is there every time you open the link.

THE SIX TOOLS — WHAT EACH ONE IS FOR
●
Dashboard
Your daily overview. Open issues, L3 alerts, checklist progress, and last audit score. Start here every shift. If there is a red alarm strip at the top — that is your first priority.
+
Issue Log
Log every maintenance issue the moment it is found. Choose the urgency level — L1 (plan it), L2 (fix today), L3 (act now). Filter by status. Mark resolved. Never let an issue disappear into someone's memory.
☑
Checklists
Daily, weekly, and monthly checks. Switch to Edit mode to customise the items for your property. Switch to Use mode to tick them off during the shift. Checklist progress shows on the Dashboard.
📄
SOP Builder
Write your maintenance procedures once. Five pre-built SOPs are included — reporting issues, L3 emergencies, planned maintenance, supplier call-outs, and room damage assessment. Edit them to match your hotel. Add your own.
📞
Contacts & Suppliers
Two sections. The Escalation Contact Card — who to call for each emergency, with backup numbers. The Supplier List — all your maintenance suppliers in one place with contact details and contract renewal dates. Fill this in before you need it.
✓
Audit
A 15-question health check across five areas: detection, logging, supplier management, team response, and prevention. Run it once now to see where the gaps are. Each result links directly to the tool that fixes it.
HOW TO GET STARTED — IN ORDER
1
Fill in your hotel information
On the Dashboard tab, enter your hotel name, maintenance manager, and number of rooms.
2
Fill in your Contacts & Suppliers
Go to the Contacts tab. Add the name and number for each supplier category. Add backup contacts. This is the most important setup step — do it before anything else fails.
3
Customise your Checklists
Go to Checklists, switch to Edit mode, and adjust the items for your property. Add areas that are relevant — pool, garden, kitchen, technical room.
4
Review and edit your SOPs
Go to SOP Builder. The five pre-built SOPs are a starting point. Edit them to reflect how your hotel actually operates. Add SOPs for anything specific to your property.
5
Run the Audit
Go to the Audit tab and answer all 15 questions honestly. The results will show you where your maintenance system is weak and link you directly to the tool that helps.
6
Use it daily
Dashboard first. Log issues as they happen. Tick off checklists each shift. That is the whole habit. Everything else builds from there.
EXPORT OPTIONS
CSV
Export to spreadsheet

Downloads your Issue Log and Supplier List as a .csv file. Does not include checklists, SOPs, or audit results.

✉ EMAIL
Send a maintenance report

Opens your mail app with a summary of open issues — L3 first, then L2. Good for end-of-shift handover or reporting to a manager off site. Plain text only.

PDF
Print the full toolkit

Prints all tabs in one document — issue log, checklists, SOPs, contacts, and audit results. All tabs render before printing so nothing is missing.

PRINT THIS SOP
Print one SOP as a clean document

Found on the SOP Builder tab. Select an SOP from the sidebar first, then click PRINT THIS SOP. Opens a clean formatted print window with just that one procedure — no buttons or navigation. Good for posting in back of house.

PRINT ALL SOPs
Print all SOPs as a procedures manual

Also on the SOP Builder tab. Prints every SOP in one document, one per page. This is your hotel's maintenance procedures manual in print form.

JSON ↓
Export SOPs as a JSON file

Found on the SOP Builder tab. Downloads all your SOPs as a .json file. Use this to back up your SOPs or transfer them to the SOP Studio tool where you can build and edit them in more detail. All steps, roles, and decision guidance transfer perfectly.

JSON ↑
Import SOPs from a JSON file

Found on the SOP Builder tab. Import a .json file exported from SOP Studio or this tool. Choose to replace all current SOPs or merge with existing ones. This is how you build SOPs in SOP Studio and bring them into the maintenance toolkit.

ONE RULE

Log everything. A maintenance issue that is not written down does not exist — until a guest finds it. The habit of logging takes two minutes. The cost of not logging can be a negative review, a repair bill, or an unhappy guest.
