THREE TOOLS IN ONE

SOP Builder
Write step-by-step procedures for your hotel. Five pre-built maintenance SOPs included. Add your own for any process — check-in, checkout, complaint handling, shift handover. Each step includes who does it, how long it takes, and what to do if it goes off-plan.

Map every touchpoint from pre-arrival to post-stay. What do guests experience at each stage? What should your team do? Where are the gaps? Use this to design the guest experience intentionally — not accidentally.
ALL EXPORT OPTIONS
JSON ↓ Export SOPs as a JSON file

Downloads all your SOPs as a .json file. Use this to back up your SOPs, share them with another hotel, or import them directly into the Maintenance Pro toolkit. The format is identical between tools so everything transfers perfectly — all steps, roles, decision guidance, and purpose fields.

JSON ↑ Import SOPs from a JSON file

Click JSON ↑ and select a .json file exported from SOP Studio or Maintenance Pro. You choose whether to replace all current SOPs or merge them with your existing ones. This is how you move SOPs between devices or between tools.

PRINT SOP Print the currently selected SOP

Opens a clean print window with just the selected SOP — no buttons, no navigation. One formatted page. Good for posting in back of house or handing to a new staff member. Select the SOP in the sidebar first.

PRINT ALL Print all SOPs as a procedures manual

Opens a clean print window with every SOP, one per page. This is your hotel's operations manual in print form. Good for a new staff member's folder, a back-of-house binder, or a handover document.

CSV Export all SOPs to spreadsheet

Downloads all SOP steps as a .csv file you can open in Excel or Google Sheets. Useful for sharing with a consultant, importing into another system, or keeping a plain text backup. Does not include formatting.

✉ Email a summary

Opens your mail app with a plain text list of all SOP titles, responsible roles, step counts, and purpose descriptions. Good for sharing an overview with a manager, owner, or consultant. Does not include full step detail — use JSON or CSV for that.

IMPORTANT — JSON ↓ IS NOT THE SAME AS BACKUP ↓
JSON ↓ exports SOPs only — for sharing with Maintenance Pro. BACKUP ↓ saves everything in this studio — SOPs, Decision Guidance, and Guest Journey. Use BACKUP ↓ to protect your data.
BACKUP ↓ Save everything in this studio

Downloads a complete backup of all three tools — SOPs, Decision Guidance categories, and Guest Journey answers. Store this file somewhere safe. If your browser data is ever cleared or you switch devices, restore from this file and lose nothing.

RESTORE ↑ Load from a full backup file

Select a studio backup file to restore SOPs, Decision Guidance, and Guest Journey. You will be asked to confirm before anything is replaced. Cannot be used to restore a SOP-only JSON file — use JSON ↑ for that.

ONE RULE

A procedure that lives only in someone's head is not a procedure — it is a risk. Write it down. When something goes wrong, update it. That is how a system improves.
