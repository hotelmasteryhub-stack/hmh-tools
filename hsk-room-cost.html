HSK Room Cost Calculator
Calculate exactly what it costs to clean each room type — by design. Labour, linen, and supplies per clean, per stay, per room night.
WHAT THIS TOOL DOES

This tool calculates the true cost of cleaning each room type in your hotel — not the monthly invoice total, but the designed cost per clean. It breaks down exactly what one departure clean, one stayover clean, and one linen change costs — by room type.

When you know the cost per clean by design, you can answer questions like: if we change our linen policy from every 7 nights to every 5 nights, what does that add to our monthly laundry bill? If we add a family room type with longer cleaning times, what does each stay cost in housekeeping?

Works with both laundry pricing models — per piece (itemised invoice) or per kg (weight-based invoice). Switch between them in the Linen tab.

WHAT EACH COST MEANS
Labour cost per departure clean
Departure time (minutes) ÷ 60 × real hourly rate. This is what one full room reset costs in staff time. Varies per room type — a family room takes longer than a single room.
Labour cost per stayover clean
Stayover time (minutes) ÷ 60 × real hourly rate. Lighter than a departure — no full linen change, no full reset. Still costs real money per service.
Linen cost per change
The cost of washing one full linen set for this room type. Per piece: sum of all items × qty per room turn. Per kg: weight of room set × kg rate. Different room types use different linen quantities.
Total cost per stay
All cleaning labour + all linen changes + supplies for the full stay. Depends on stay length, number of stayover cleans, and number of linen changes — all set per room type in the Room Types tab.
HSK cost per room night
Total stay cost ÷ stay length. This is the number that feeds into your room cost and pricing calculations. Compare it to your monthly invoice total ÷ room nights sold to check alignment.
HOW TO USE — FOUR STEPS
1
Setup — enter staff rate and laundry model
Enter your gross hourly rate and employer cost multiplier. Choose your laundry pricing model — per piece or per kg. This feeds all calculations.
2
Room Types — set cleaning times and stay policy per room type
Add each room type — Small, Double, Family, Dorm Bed. For each: departure clean time, stayover clean time, typical stay length, number of stayover cleans per stay, number of linen changes per stay.
3
Linen & Supplies — set prices and quantities
Per piece: enter price per linen item and how many of each go into one room set per room type. Per kg: enter your rate per kg and the weight of one room set per room type. Also set supply cost per room night.
4
Results — read and compare
Full breakdown per room type — labour per clean, linen per change, supplies, total per stay, HSK cost per room night. Export as PDF for your pricing review.
