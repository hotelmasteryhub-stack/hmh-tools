WHAT THIS TOOL DOES

This tool calculates the true cost of cleaning each room type — by design. It breaks down exactly what one departure clean, one stayover clean, and one linen change costs per room type, based on your actual cleaning times, linen configuration, and staff rate.

Linen is structured the way it actually works — bed sets (single, double, king, bunk) each with their own contents and price, plus room-level items like bath mats and pool towels. Each room type specifies how many of each bed set it uses, so a family room with 2 single beds and 1 double bed calculates correctly.

Works with both laundry pricing models — per piece (individual item prices) or per kg (weight of one complete set). Switch in Setup.

HOW TO USE — FIVE STEPS
1
Setup — staff rate and laundry model
Enter gross hourly rate and employer cost multiplier → real cost per hour auto-calculates. Choose Per Piece or Per Kg laundry pricing.
2
Room Types — cleaning times and stay policy
Add each room type. Set departure time, stayover time, stay length, number of stayover cleans, and number of linen changes per stay.
3
Linen — bed sets and room items
Define your bed set types (Single, Double, King etc.). Per piece: enter each item inside the set and its price. Per kg: enter the weight of one complete set. Add room-level items (bath mat, pool towel) separately. Then for each room type, enter how many of each bed set it uses.
4
Supplies — monthly invoice ÷ nights sold
Enter your monthly supply invoice total and room nights sold. Supply cost per room night calculates automatically. This is a design estimate — one figure for all room types.
5
Results — full cost breakdown per room type
Labour per clean, linen per change, supplies, total per stay, HSK cost per room night. Linen policy impact table shows the cost of 1, 2, or 3 linen changes per stay.
