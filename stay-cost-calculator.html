Stay Cost Calculator
What does a full stay actually cost per room type? Base hotel costs + extra cleaning labour + linen = total stay cost and required nightly rate.
WHAT THIS TOOL DOES

The Room Cost Calculator tells you what one sold room costs per night at a given occupancy. But different stays cost different amounts — a 3-night stay with a departure clean costs less than a 14-night long stay with mid-stay services and linen changes.

This tool takes the base cost per room night from the Room Cost Calculator and adds the extra cleaning labour (stayover services and linen change labour) and linen costs specific to each stay type. The result: a total cost per stay and a required nightly rate for each room type.

Linen is calculated the right way — per bed set type (single, double, king) with actual item prices or weights, so a family room with 2 single beds and 1 double calculates correctly.

HOW TO USE — FOUR STEPS
1
Setup — enter base costs and labour rate
Enter hotel name and the base cost per sold room night from the Room Cost Calculator. Enter gross hourly rate and employer multiplier → real cost per hour for cleaning labour. Choose laundry mode (per piece or per kg) and enter kg rate if applicable.
2
Linen — define bed set types and room items
Add each bed set type used in your hotel (Single, Double, King etc.). Per piece: enter each item inside the set and its price. Per kg: enter the weight of one complete set. Add room-level items (bath mat, pool towel) separately.
3
Room Types — define each stay type
Add each room type. Set stay length, departure clean time, stayover clean time, number of stayover cleans, number of linen changes per stay, bed set allocation (how many of each set per room), and margin override if different from global.
4
Results — read required price per night
Full cost breakdown per room type — base cost, extra cleaning labour, linen cost, total stay cost, required revenue, required price per night. Compare different stay types side by side.

ONE RULE

A 14-night long stay is not 14 times a 1-night stay. The cleaning schedule is different, the linen changes are different, and the required nightly rate is different. Price by stay type — not by assumption.
