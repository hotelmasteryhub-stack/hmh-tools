HOTEL MASTERY HUB
Room Cleaning Checklist
Three checklists in one — Checkout Clean, Stayover Service, and Inspection. Archive every completed shift.
HOW TO USE
1
Set the room details — enter the room number, housekeeper name, and today's date at the top of each checklist tab before starting.
2
USE mode — tick items as you clean. Click any row or the checkbox. Ticked items show with strikethrough. Use RESET TICKS to start fresh for the next room.
3
EDIT mode — customise the checklist for your hotel. Add, rename, or remove items and sections. Changes save automatically.
4
SAVE TO HISTORY — after completing a checklist, click SAVE TO HISTORY at the bottom of the tab. The completed state (room number, housekeeper, date, ticked items) is archived. History keeps the last 90 days.
5
HISTORY tab — browse all archived checklists from the last 90 days. Expand any entry to see the full item-by-item tick record. Download full history as CSV.
EXPORTS — WHAT EACH BUTTON DOES
PDF — prints the current tab only. Checkout Clean prints only the checkout checklist. Never prints all tabs at once.
✉ Email header button — sends the checklist content of the current tab. Room number, housekeeper, date, and all items included.
✉ Email This Checklist button — at the bottom of each tab — same as above but placed for convenience.
DOWNLOAD THIS PAGE ↓ — downloads the current checklist tab as a clean printable PDF.
SAVE TO HISTORY — archives the current completed checklist (room, housekeeper, date, all tick states) to the 90-day history.
CSV — exports all three checklists (items only, not tick states) as a spreadsheet.
BACKUP ↓ — saves everything: all checklist content, all tick states, room/housekeeper/date fields, and full history as a .json file.
RESTORE ↑ — loads a backup file. Confirms before replacing. Shows the exported date.
ONE RULE

A checklist that is not saved is a cleaning that cannot be proved. Save every completed room to history. When a guest disputes the condition of a room, you will have the record.
