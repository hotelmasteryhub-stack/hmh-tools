HOTEL MASTERY HUB
Room Cleaning Checklist
Task checklists for checkout cleans, stayover services, and room inspection. Edit mode to customise. Use mode to tick off during work.
HOW TO USE
1
Edit mode — customise the checklists
Switch to EDIT mode using the button at the top of each tab. Add, rename, or remove items per section to match your hotel's standard. Your changes save automatically.
2
Use mode — tick items during work
Switch to USE mode. Tick each item as you complete it. Completed items turn green with a strikethrough. Reset ticks when starting a new room.
3
Download each checklist separately
Use DOWNLOAD THIS PAGE ↓ on each tab to print that checklist as a standalone reference card.
EXPORT OPTIONS
CSVAll checklists

Downloads all three checklists — checkout, stayover, and inspection — with all sections and items.

✉Email — current tab

Sends the current checklist as plain text.

PDFPrint — current tab

Prints the current checklist only. Use DOWNLOAD THIS PAGE ↓ at the bottom of each tab.

BACKUP ↓Save all data as JSON

Complete backup of all three checklists and all customisations.

RESTORE ↑Load from backup

Select a .json backup. Confirms before replacing.
