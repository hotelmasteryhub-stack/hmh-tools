Housekeeper Onboarding Builder
Hotel Mastery Hub · Housekeeping
WHAT THIS TOOL DOES

This tool helps you build a structured onboarding programme for every new housekeeper. It covers access they need, what they need to learn, and tracks their progress from observation to independent work.

It works in two modes — Edit and Use. Build it once for your hotel, then use it for every new starter.

TWO MODES
EDIT mode
Customise everything for your hotel. Add, remove, or rewrite tasks to match your standards. Edit the access checklist. Add new task categories. This is your template — set it up once and reuse it for every new starter.
USE mode
Track the trainee's progress. Set the stage per task — Observation, Guided, Independent, or Signed off. Add buddy sign-off and date. Progress bar updates automatically. When all tasks are signed off, the document is ready for final sign-off.
HOW TO GET STARTED
1
Go to EDIT mode
Work through each section. Edit task descriptions to match your hotel's exact standards. Delete tasks that don't apply. Add tasks that are missing.
2
Fill in the Access Checklist
Add how access is requested at your hotel. Some items may not apply — mark them N/A.
3
Assign a buddy
Enter the buddy name in the trainee profile before the new starter arrives. Brief the buddy on how the tool works.
4
Switch to USE mode
When the new starter arrives, switch to Use mode. Work through tasks together. Set the stage for each one as training progresses.
5
Complete sign-off
When all tasks are signed off, fill in Section D and save as PDF for your records.
EXPORT OPTIONS
PDF
Prints the onboarding document — trainee profile, access checklist, all tasks with progress, and sign-off section. Use for printed records or to share with the buddy.
CSV
Downloads all tasks with their current stage and sign-off status. Useful for keeping a spreadsheet record of completed onboardings per staff member.
✉ EMAIL
Sends a plain text progress summary — trainee name, how many tasks are signed off, and which ones are still pending. Good for updating a manager remotely.
BACKUP ↓
Downloads everything — profile, access checklist, all tasks with progress and notes, and the final sign-off — as a single JSON file. Store it safely; restore from it if your browser data is cleared or you switch devices.
RESTORE ↑
Loads a backup file. You'll be asked to confirm before it replaces anything currently on screen.
ONE RULE

A task is only signed off when the trainee can do it independently — not just when they have been shown it. There is a difference. This document exists to track that difference.
