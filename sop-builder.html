WHAT THIS TOOL DOES

An SOP — Standard Operating Procedure — is a written step-by-step process for how something is done in your hotel. When a procedure exists only in someone's head, it disappears when they leave. When it is written down, anyone can follow it.

This builder lets you write, edit, and print your hotel's procedures. Each SOP has a title, responsible role, steps with decision guidance, and a last-reviewed date. Pre-built maintenance SOPs are included as a starting point.

TWO MODES
✎ EDIT mode
Write and edit your SOPs. Add steps, fill in who does what and when, add decision guidance for when things go off-plan. Select SOPs from the sidebar list on the left.
📄 VIEW mode
Shows all your SOPs as clean formatted documents — no editable fields, no buttons. Use this before printing or sharing. Print this SOP and Print all SOPs buttons work from this mode.
HOW TO GET STARTED
1
Select an SOP from the sidebar
Five pre-built maintenance SOPs are ready to edit. Click any one to open it.
2
Edit the title, role, and purpose
Update the SOP name, who is responsible, and a one-sentence description of what it covers.
3
Edit or add steps
Each step has: what happens, who does it, how long it takes, and what to do if it goes off-plan. The off-plan column is the most important — it answers the question staff ask when something unexpected happens.
4
Add your own SOPs
Click + NEW SOP in the sidebar. Name it, assign a role, and build the steps. Think: check-in, checkout, complaint handling, room inspection, shift handover.
5
Switch to VIEW mode and print
Click VIEW to see the clean formatted version. Use PRINT THIS SOP for one procedure, or PRINT ALL SOPs for the full set. Each SOP prints on its own page.
EXPORT OPTIONS
PRINT THIS SOP Print one SOP as a clean document

Opens a print window with just the currently selected SOP. One clean page. Good for posting in back of house or giving to a new staff member.

PRINT ALL SOPs Print all SOPs as a full procedures manual

Prints every SOP in one document, one per page. Good for creating a printed operations manual for the hotel.

CSV Export all SOPs to spreadsheet

Downloads all SOP steps as a .csv file. Useful for sharing with a consultant, importing into another system, or keeping a backup.

✉ EMAIL Email a summary of your SOPs

Sends a plain text list of all SOP titles, responsible roles, and step counts. Good for sharing a quick overview with a manager or consultant.

ONE RULE

A procedure that exists only in someone's head is not a procedure — it is a risk. Write it down. Review it when something goes wrong. Update it when the process changes.
