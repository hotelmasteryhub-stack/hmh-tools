HOTEL MASTERY HUB
Team KPI Tracker
Weekly performance tracking for every department. Each manager tracks their own team. Everything flows into one overview.
WHAT THIS TOOL DOES

This tool lets each department track their own weekly KPIs — the numbers that tell you if your team is performing. Front Desk tracks review scores and upsell. Housekeeping tracks overhour % and rooms on time. F&B tracks covers and revenue per cover.

Each department manager uses this tool on their own device. They set their own KPIs and targets in Edit mode. Every Monday they switch to Use mode, enter last week's actuals, and see green or red against target.

Three departments come pre-filled with realistic example data — Front Desk, Housekeeping, and F&B. The data shown is example only. Replace it with your own in Edit mode.

TWO MODES — EDIT AND USE
✎ EDIT mode
Customise your department. Rename it. Add KPIs that matter to your team. Delete ones that don't apply. Set targets. Set the unit — %, €, or number. Do this once when you set up. Update when targets change.
☑ USE mode
Every Monday, enter last week's actuals. Status shows green (on target), amber (within 10% of target), or red (off target). The 8-week history table below shows your trend over time.
HOW TO GET STARTED
1
Choose your department tab
Three departments are pre-filled as examples — Front Desk, Housekeeping, and F&B. Click the tab for your department.
2
Switch to EDIT mode and customise
Rename the department, edit KPI names, set your targets, choose units. Delete KPIs that don't apply. Add your own. The example data is a starting point — make it yours.
3
Every Monday — switch to USE mode
Enter the week label (e.g. "W01 Jan") and last week's actuals for each KPI. Status calculates automatically. History table updates.
4
Check the Overview tab
The Overview tab shows all departments and all KPIs at a glance — green, amber, red. One view. Share it as a PDF for your weekly team meeting.
5
Add your own department
Click + ADD DEPARTMENT in the tab bar. Name it, add your KPIs, set targets. Works for any team — spa, maintenance, reception, events.
EXPORT OPTIONS
CSVExport all KPI data to spreadsheet

Downloads all departments, all KPIs, targets, and 8-week history as a .csv file. Good for sharing with a GM or owner, or keeping an archive.

✉Email a weekly summary

Sends a plain text summary of all departments — this week's actuals vs targets, with green/red status. Good for a Monday morning update to the GM or property owner.

PDFPrint all departments

Prints the full tracker — all departments, all KPIs, current week and 8-week history. Good for a weekly team meeting or printed report for the owner.

BACKUP ↓Save everything as a JSON file

Downloads a complete backup of all departments, KPIs, targets, and all weekly history. Save it somewhere safe. If your browser data is ever lost, you can restore from this file.

RESTORE ↑Load from a backup file

Select a .json backup file to restore all your data. You will be asked to confirm before anything is replaced. Use this if you switch devices or your browser data is cleared.

ONE RULE

A KPI that is never reviewed is just a number. Review it every Monday. If it is red two weeks in a row — act. That is the point of tracking it.
